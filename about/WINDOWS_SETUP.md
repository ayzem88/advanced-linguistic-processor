# تعليمات التثبيت والتشغيل على Windows

## المتطلبات الأساسية

### 1. Python
- تحميل Python 3.8 أو أحدث من [python.org](https://www.python.org/downloads/)
- تأكد من اختيار "Add Python to PATH" أثناء التثبيت

### 2. التحقق من التثبيت
افتح Command Prompt وشغل:
```cmd
python --version
pip --version
```

## التثبيت

### الطريقة الأولى: التشغيل التلقائي
1. انقر نقراً مزدوجاً على `run_windows.bat`
2. سيقوم بتثبيت المكتبات المطلوبة تلقائياً

### الطريقة الثانية: التثبيت اليدوي
افتح Command Prompt في مجلد البرنامج:
```cmd
pip install -r requirements.txt
```

## التشغيل

### الطريقة الأولى: ملف bat
انقر نقراً مزدوجاً على `run_windows.bat`

### الطريقة الثانية: سطر الأوامر
```cmd
python main.py
```

## دعم اللغة العربية

### Windows 10/11
- دعم العربية مدمج بالكامل
- الخطوط العربية متوفرة افتراضياً

### Windows 7/8
قد تحتاج لتثبيت حزمة دعم اللغة العربية من:
- Settings → Time & Language → Language
- أضف اللغة العربية

## حل المشاكل الشائعة

### 1. خطأ "Python is not recognized"
- أعد تثبيت Python وتأكد من اختيار "Add to PATH"
- أو أضف Python للـ PATH يدوياً

### 2. خطأ "No module named PyQt6"
```cmd
pip install PyQt6
```

### 3. مشاكل الترميز العربي
أضف هذا في بداية الكود:
```python
# -*- coding: utf-8 -*-
```

### 4. الخطوط العربية لا تظهر
- تأكد من تثبيت خطوط عربية (مثل: Arial أو Tahoma)
- قد تحتاج لتغيير الخط في الإعدادات

## الملفات المهمة

```
Albhthya/
├── main.py              # الملف الرئيسي للتشغيل
├── run_windows.bat      # ملف التشغيل السريع
├── requirements.txt     # قائمة المكتبات المطلوبة
├── data/               # قاعدة البيانات والكتب
├── ui/                 # واجهة المستخدم
└── db/                 # إدارة قاعدة البيانات
```

## الأداء على Windows

الكود محسّن للعمل على Windows:
- ✓ دعم كامل لـ UTF-8
- ✓ مسارات متوافقة مع Windows
- ✓ واجهة رسومية native
- ✓ أداء ممتاز مع قواعد البيانات الكبيرة

## الدعم الفني

إذا واجهت مشاكل:
1. تأكد من تثبيت Python 3.8+
2. تأكد من تثبيت جميع المكتبات
3. تحقق من وجود ملف قاعدة البيانات في `data/shamela.db`
